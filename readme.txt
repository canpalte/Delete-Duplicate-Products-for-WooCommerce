Easily find and remove duplicate products in your WooCommerce store.

== Description ==

This plugin provides a simple and effective solution for managing duplicate products in WooCommerce. With it, you can:

<ul>
  <li><strong>Detect:</strong> Automatically find duplicate products based on title, SKU, or content.</li>
  <li><strong>Review:</strong> View a complete list of all duplicate products found.</li>
  <li><strong>Remove:</strong> Safely delete duplicate products while keeping the original version.</li>
</ul>

<p><strong>Key Features:</strong></p>
<ul>
  <li>Intuitive interface for duplicate management</li>
  <li>Customizable search options</li>
  <li>Safe deletion process</li>
  <li>Compatible with the latest WooCommerce version</li>
</ul>

== Installation ==

<h3>Using WordPress Plugin Installer</h3>
<ol>
  <li>Go to the "Plugins" section in your WordPress dashboard.</li>
  <li>Search for "Delete Duplicate Products for WooCommerce".</li>
  <li>Click "Install Now".</li>
</ol>

<h3>Manual Installation</h3>
<ol>
  <li>Download the plugin ZIP file.</li>
  <li>Upload the ZIP file to the "/wp-content/plugins/" directory on your WordPress site.</li>
  <li>Activate the plugin from the "Plugins" menu.</li>
</ol>

<h3>Using the Plugin</h3>
<ol>
  <li>Go to Duplicate Products on Menu</li>
  <li>Review the list of duplicate products found</li>
  <li>Select and remove the duplicates you want</li>
</ol>
